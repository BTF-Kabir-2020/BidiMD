# Security

**BidiMD** does not patch VS Code or Cursor workbench files. It only uses the public Extension API (`markdown.styles` and commands).

## Reporting a vulnerability

Please open a private report or an issue at:

https://github.com/BTF-Kabir-2020/BidiMD/issues

Do **not** include Personal Access Tokens, Open VSX tokens, or other secrets in issues or screenshots.
